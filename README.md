**e-plantShopping** is a web-based application designed to provide users with a smooth and convenient online plant shopping experience. The application allows users to browse a variety of plants, view product details, and simulate an e-commerce workflow for purchasing plants.

This project focuses on building a clean, user-friendly interface and implementing core shopping features such as product listing, cart management, and navigation. The goal of **e-plantShopping** is to demonstrate practical frontend and backend development concepts used in modern web applications.

### Key Features
- Browse a collection of plants available for purchase
- View detailed information for each plant
- Add and remove plants from the shopping cart
- Simple and responsive UI design
- Structured and modular project architecture

### Purpose
The **e-plantShopping** repository is created as a learning project to apply web development skills and understand how an online shopping platform works at a fundamental level.
